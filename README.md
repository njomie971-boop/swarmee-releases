# SWARMEE — Releases (alpha)

Binaries only — the source lives in a separate private repository.

## Installer / jouer

1. Télécharge **SwarmeeLauncher.exe** depuis la dernière release.
2. Lance-le : il installe le jeu dans `%LOCALAPPDATA%\SWARMEE`, vérifie
   l'intégrité (SHA-256) et te proposera automatiquement les mises à jour.
3. Windows SmartScreen peut avertir (binaire non signé) : « Informations
   complémentaires » → « Exécuter quand même ».

Chaque release contient : le jeu zippé (`SWARMEE-vX.Y.Z-win64.zip`), le
launcher, et `latest.json` (manifeste de mise à jour : version + SHA-256).
